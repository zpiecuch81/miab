Introduction

MIAB, Message In A Bottle, is an application that allows to leave messages in real places. Anyone who would like to take the message need to be in the same place.
Basic usage of the program is following:

  1. User writes a message and leaves it.
  2. When other (or the same) MIAB user walks in the same place with "looking for bottles option enabled", message is found and can be read.

FAQ

  1. Can I destroy my message?
    No, but if you will be the first person that finds it, then no one else will get it.
  2. How many people will get my message?
    Each message can be taken only once.
  3. How long will my message be "active"?
    Once message is left, it's there till someone finds it.
  4. Can you tell who left specific message?
    No. Once message is left, there is no connection to its creator in MIABs database. If you like you can write your email, name, facebook data etc.
  5. Can I track my message?
    No. Once message is left there is no connection to its creator, so it can't be tracked.
  6. What is the direction and speed of thrown bottles?
    Speed and direction of flowing is generated at the moment of throwing a bottle.
  7. Are messages encrypted?
    No.
  8. Can I use the program inside buildings?
    Generally yes. However because GPS accuracy is low, it might be impossible to leave bottles inside buildings.
  9. Why do I need to have GPS enabled?
    Location is needed to leave bottles in proper place. Also when looking for bottles location is required. GPS data are not used to anything else.
  10. I left a message and next day I can't find it in the same place. Why?
    First: it is possible that someone else already found it. Second: although we follow the best practices when gathering GPS data, accuracy of GPS systems varies depending on weather, producer and other parameters. Unfortunately this may cause the message to be in slightly different position than you think you left it. Consider checking circle area with up to 10 meter radius.
